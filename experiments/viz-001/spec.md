# viz-001: Interactive Particle System

## Purpose
Create an engaging, interactive particle system visualization to demonstrate the experiment framework's capabilities.

## Method
- Canvas-based rendering using requestAnimationFrame for smooth animation
- Particles react to mouse/touch input
- Physics simulation with velocity, friction, and collision detection

## Input/Output
- Input: Mouse/touch position
- Output: Real-time rendered particles with trails

## Constraints
- Client-side only (no external APIs)
- Single-file demo component
- Performance: 60fps on modern browsers
- Max 200 particles
