# weird-002: Glitch Effect Generator

## 목적
텍스트에 디지털 글리치 효과를 적용하여 사이버펑크/레트로 스타일의 시각 효과를 생성합니다.

## 방법
- HTML5 Canvas를 사용하여 텍스트 렌더링
- RGB 채널 분리로 색상 왜곡 효과 구현
- 픽셀 정렬 알고리즘으로 데이터 손상 효과 시뮬레이션
- 스캔라인 및 노이즈 오버레이로 CRT 모니터 효과 추가

## 입력/출력
- 입력: 텍스트 문자열, 글리치 강도/타입 설정
- 출력: 글리치 처리된 이미지 (PNG 다운로드)

## 제약
- 클라이언트 사이드 전용 (Canvas API)
- 단일 파일 데모 컴포넌트
- 텍스트 기반 입력만 지원 (이미지 업로드 제외)
