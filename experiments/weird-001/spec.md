# weird-001: Gravity Reversal

## 목적
마우스/터치 위치를 기준으로 중력 방향이 반전되는 독특한 인터랙션을 보여줍니다.

## 방법
- Canvas 기반 물리 시뮬레이션
- 파티클이 마우스/터치 위치를 따라감
- 중력 반전 버튼으로 중력 방향 180도 변경
- 시각적 피드백: 방향 표시, 색상 변화

## 입력/출력
- 입력: 마우스/터치 위치, 중력 반전 버튼 클릭
- 출력: 중력 반전 애니메이션, 시각적 피드백

## 제약
- 클라이언트 사이드 전용 (외부 API 없음)
- 단일 파일 데모 컴포넌트
- 성능: 60fps on modern browsers
- 접근성: 키보드/마우스 조작 가능한 기본 UI

## 버그 수정 (Issue #31)
- 파티클 초기화 로직 개선
- 애니메이션 루프 의존성 문제 해결
- 캔버스 리사이즈 처리 강화
