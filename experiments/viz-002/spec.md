# viz-002: Fractal Visualization (Mandelbrot Set)

## 목적
만델브로트 집합을 캔버스에 렌더링하여 무한한 복잡성의 수학적 아름다움을 시각화합니다.

## 방법
- HTML5 Canvas를 사용한 픽셀 단위 렌더링
- 복소수 평면에서 z = z² + c 반복 계산
- 색상 매핑: 반복 횟수에 따른 그라데이션
- 줌/패닝: 복소수 평면 좌표 변환

## 입력/출력
- 입력: 줌 레벨, 중심 좌표, 색상 팔레트, 최대 반복 횟수
- 출력: 렌더링된 만델브로트 집합 이미지

## 제약
- 클라이언트 사이드 전용 (Canvas API)
- 성능: requestAnimationFrame 사용
- 단일 파일 데모 컴포넌트
- 접근성: 키보드/마우스 조작 가능
