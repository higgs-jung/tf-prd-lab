# feature-004: 오늘의 실험 (Today's Experiment)

## 목적
홈페이지에 매일 바뀌는 추천 실험을 표시하여 사용자 참여를 유도합니다.

## 방법
- 날짜 기반 해시로 매일 다른 실험 선택 (24시간 동안 동일)
- 랜덤 실험 뽑기 버튼으로 즉시 다른 실험 탐색
- 카테고리별 색상 구분으로 시각적 차별화
- 통계 표시로 전체 실험 규모 강조

## 입력/출력
- 입력: 사용자의 "랜덤 실험 뽑기" 버튼 클릭
- 출력: 선택된 실험 정보 및 링크

## 제약
- 클라이언트 사이드에서 날짜 계산 (서버 의존성 없음)
- localStorage 없이 순수 날짜 기반 로직
- 반응형 디자인 (모바일/데스크톱)
