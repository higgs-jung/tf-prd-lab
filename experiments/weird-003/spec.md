# weird-003: Audio Feedback Delay Toy

## 목적
오디오 입력을 짧은 딜레이 라인에 넣고 피드백을 걸어, 간단한 에코/자가발진(주의) 텍스처를 “장난감”처럼 빠르게 탐색합니다.

## 방법
- Web Audio API로 `DelayNode` + `GainNode`(feedback) + `BiquadFilterNode`(톤/안전) 구성
- Dry/Wet 믹스와 feedback 값을 실시간 슬라이더로 조절
- 출력 레벨을 제한(컴프/리미터 또는 고정 게인)하여 귀/스피커 보호

## 입력/출력
- 입력: 마이크(옵션) 또는 간단한 테스트 톤/노이즈
- 출력: 브라우저에서 재생되는 딜레이/피드백 사운드

## 제약
- 헤드폰 권장, 기본값은 안전한 피드백 범위로 시작
- 브라우저 권한(마이크) 및 자동재생 정책에 영향 받음
